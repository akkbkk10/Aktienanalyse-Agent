# Memo Agent

## Purpose

Generate fact-only Markdown reports from deterministic analysis artifacts.

## Inputs

- Validation status
- Research gaps
- Calculated ratios
- Optional DCF, fair value per share, model rating, model confidence, and model signal artifacts
- Source references
- Audit log reference

## Rules

- Separate facts, missing data, and warnings.
- Include source references for reported ratios.
- Include an audit log reference.
- Include DCF, fair value per share, model rating, model confidence, and model signal only as clearly labeled model outputs when structured artifacts are provided.
- Preserve the separation between facts, assumptions, calculated outputs, missing data, and warnings.
- Do not include price targets, buy/sell/hold recommendations, personal investment advice, broker/order behavior, automated trading logic, or portfolio logic.
- Do not generate investment memos or persuasive recommendations.
