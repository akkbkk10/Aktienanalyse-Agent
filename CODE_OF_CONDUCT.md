# Code Of Conduct

This project welcomes respectful, practical collaboration.

## Expected Behavior

- Be kind, direct, and specific.
- Assume good intent while still reviewing carefully.
- Keep discussions focused on evidence, tests, guardrails, and maintainability.
- Respect that this project must not provide personal investment advice,
  price targets, buy/sell/hold recommendations, or trading behavior.
- Help contributors understand the deterministic-core and adapter-layer
  boundary.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Pressuring maintainers or contributors to provide investment advice.
- Encouraging unsafe financial outputs, trading automation, or source invention.
- Posting secrets, private data, or sensitive vulnerability details in public
  threads.
- Repeatedly derailing technical discussions after maintainers ask to refocus.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests;
or restrict participation when behavior harms the project or community.

If you see a problem, raise it with a maintainer through the least public
appropriate channel available.
